断言是判断返回值是否满足预期结果的方法

AssertTrue/AssertFalse    断言是否为真值还是假
\AssertEquals    判断输出是否和预期的相等
\AssertGreaterThan    断言结果是否大于某个值，同样的也有LessThan(小于),GreaterThanOrEqual(大于等于)，
\LessThanOrEqual(小于等于).
\AssertContains    判断输入是否包含指定的值
\AssertType    判断是否属于指定类型
\AssertNull    判断是否为空值
\AssertFileExists    判断文件是否存在
\AssertRegExp    根据正则表达式判断

例1：写一个类，含有一个静态方法，此方法含有一个数组参数，方法可以把参数数组的所有的健对应的标准数据类型生成新的数组，测试此方法的正确性

cat Demo1.php

<?php
class classDemo
{
    //获取数组中所有标准类型的值
    public static function getArrayValue(array $array)
    {
        $return = array();
        foreach($array as $item){
            if(is_array($item)){
                $return = array_merge($return,self::getArrayValue($item));
            }else if(is_object($item) || is_resource($item) || is_null($item)){
                continue;
            }else{
                array_push($return,$item);
            }
        }

        return array_unique($return);
    }
}

class Demo1 extends PHPUnit_Framework_TestCase
{
    public function testWindow()
    {
        //原始数据
        $return = array(
            1,2,3=>array(1,3,5,'a'),'a'=>NULL,'b'=>new stdClass(),'c'=>@fopen('./CacheTest.php','r')
        );

        //实际值
        $data = classDemo::getArrayValue($return);
        //预期值
        $shouldBeResult = array(1,2,3,5,'a');
        //取差集
        $interact = array_diff($shouldBeResult,$data);
        //差集为空则验证通过
        $this->assertEquals(count($interact),0);
    }
}
